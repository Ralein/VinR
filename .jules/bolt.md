## 2026-04-25 - Concurrent API Execution with asyncio.gather\n**Learning:** In asynchronous services calling multiple external HTTP endpoints, sequential awaits create latency bottlenecks (N+1-like). Memory constraints explicitly indicate that while AsyncSession prevents database concurrency, independent HTTP requests via httpx.AsyncClient should be gathered.\n**Action:** Always search for sequential awaits of independent network calls in FastAPI services and wrap them in `asyncio.gather` to parallelize execution.
